# App Store Screenshots Guide

## Required Screenshot Sizes

You need screenshots for at least ONE of these device sizes:
- **6.7" Display (iPhone 15 Pro Max)**: 1320 x 2868 pixels
- **6.5" Display (iPhone 11 Pro Max)**: 1242 x 2688 pixels
- **5.5" Display (iPhone 8 Plus)**: 1242 x 2208 pixels

**Recommended:** Provide 6.7" screenshots (latest iPhones)

## Number of Screenshots
- **Minimum:** 1 screenshot
- **Maximum:** 10 screenshots
- **Recommended:** 3-5 screenshots showing key features

## How to Capture Screenshots

### Option 1: Using iOS Simulator (Easiest)

1. **Launch your app in simulator:**
   ```bash
   cd /Users/s0m13i5/linus/multi-media/mobile
   npm run ios
   ```

2. **Select the right device:**
   - In Simulator, go to: Device → Choose a device
   - Select: iPhone 15 Pro Max (for 6.7" screenshots)

3. **Capture screenshots:**
   - Navigate to each screen you want to capture
   - Press: `Cmd + S` to save screenshot
   - Screenshots save to: Desktop

4. **Verify size:**
   - Right-click screenshot → Get Info
   - Should be: 1320 x 2868 pixels

### Option 2: Using Real iPhone

1. Connect iPhone 15 Pro Max (or similar large iPhone)
2. Build and run app on device
3. Navigate to each screen
4. Press: Side button + Volume Up (simultaneously)
5. Screenshots save to Photos app
6. AirDrop to Mac

### Option 3: Using Design Tools

1. Use Figma, Sketch, or Photoshop
2. Create canvas: 1320 x 2868 pixels
3. Place your app screenshots
4. Add text overlays to highlight features (optional)
5. Export as PNG

## Tips for Great Screenshots

✅ **Do:**
- Use the app in light mode for better visibility
- Show actual content, not loading screens
- Ensure text is readable
- Use the most attractive screens
- Add text overlays to highlight features (optional)
- Show the app in action (playing music, showing videos)

❌ **Don't:**
- Show empty states or error messages
- Use blurry or pixelated images
- Include personal information
- Show copyrighted content without permission
- Use screenshots from different device sizes together

## Quick Screenshot Checklist

Before uploading to App Store Connect:
- [ ] Correct size: 1320 x 2868 pixels (or appropriate size)
- [ ] File format: PNG or JPEG
- [ ] All screenshots from same device size
- [ ] 3-5 screenshots showing key features
- [ ] No personal/sensitive information visible
- [ ] App looks good and professional
- [ ] Screenshots are in the order you want them displayed

## Upload Process

1. Go to: https://appstoreconnect.apple.com/apps/6758027593
2. Click: "1.0 Prepare for Submission"
3. Scroll to: "App Preview and Screenshots"
4. Select device size: "6.7" Display"
5. Drag and drop your screenshots
6. Rearrange order if needed
7. Click "Save"
